CAWeb Theme Icon Font Library

The CAWeb Theme icon font library includes 635 icons you can use in your site.

To insert an icon onto your page, insert an empty span tag with the class name for the icon you wish to use, for example:

<h2><span class="ca-gov-icon-arrow-down"></span> Heading</h2>

will appear as:

Heading

When icons are used with H1 to H5 headings, styles for font-size and color are included. When used anywhere else you'll need to add styles for the font-size and color yourself. Icon fonts can be styled using the same css rules used to style any other text on the page. For example color: and font-size: chnage the color and size of the icon.
